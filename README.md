# CampusSecondhand
基于servlet+tomcat+mysql的二手物品交易平台
#
登录主页面，实现功能有：  
##### &emsp;&emsp;1主页商品显示  
##### &emsp;&emsp;2关键字搜索商品  
##### &emsp;&emsp;3点击查看商品详细信息  
##### &emsp;&emsp;4站内联系商品卖家，可互动发送聊天消息  
##### &emsp;&emsp;5上传商品，及商品分类  
##### &emsp;&emsp;6收藏商品  
##### &emsp;&emsp;7购买历史记录  
##### &emsp;&emsp;8tip：  
&emsp;交易方式非网上付款，而是确认购买商品后，与卖家联络沟通，沟通达成共识后，线下见面交易。  
&emsp;即是保障了校园内物品交易的安全性，也提供了二手物品的展示，提供了平台
